# Ansible Collection - fahcsim.grafana_stack
### This collection has a role for each of the following:
- Prometheus
- Loki
- Prometheus Node Exporter
- Promtail

Current supported distributions are Debian 10/11 and Ubuntu 20/22.

### Vars
##### Grafana

##### Prometheus

##### Loki
version: 2.5.0
##### Prometheus Node Exporter
version: 1.3.1
##### Promtail
